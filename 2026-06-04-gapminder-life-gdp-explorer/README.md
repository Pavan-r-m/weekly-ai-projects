# 🌍 Gapminder Life Expectancy & GDP Correlation Explorer

A data analysis and visualization project that digs into the famous **Gapminder dataset** — 142 countries tracked across 12 time points (1952–2007) — to reveal how GDP, population, and life expectancy evolved around the world.

---

## What It Does

| Analysis | Output |
|---|---|
| Animated bubble chart (GDP × Life Expectancy over time) | `outputs/bubble_chart.html` |
| Regional heatmap — avg life expectancy by continent & decade | `outputs/regional_heatmap.png` |
| Scatter / pairplot matrix of key variables | `outputs/scatter_matrix.png` |
| K-Means clustering of countries into development groups | `outputs/clusters.png` |
| Top 15 countries by life expectancy gain (1952→2007) | `outputs/top_gainers.png` |
| Per-continent summary statistics | `outputs/summary_stats.csv` |

---

## Tech Stack

- **pandas** — data wrangling and aggregation
- **plotly** — interactive animated bubble chart (Hans Rosling–style)
- **seaborn + matplotlib** — static publication-quality charts
- **scikit-learn** — K-Means clustering + PCA dimensionality reduction

## How It Works

### 1. Animated Bubble Chart
Replicates Hans Rosling's famous TED-talk visualization using Plotly Express. Each bubble is a country; size = population; color = continent; the slider animates through decades. The x-axis is log-scaled to spread out low-GDP countries.

### 2. Regional Heatmap
Groups years into decades, computes the mean life expectancy per continent, and renders a Seaborn heatmap. Darker = longer lives — the color gradient makes the global improvement unmistakable.

### 3. Scatter Matrix
A Seaborn pairplot on the most recent year's data with `log(GDP)` and `log(population)` — useful for spotting whether population size really predicts wealth or longevity (spoiler: it mostly doesn't).

### 4. K-Means Clustering
Standardizes three features (life expectancy, GDP per capita, population), runs K-Means with k=4, and visualizes clusters in 2-D via PCA. Clusters emerge naturally as *Developing, Emerging, Advanced, Frontier* — a data-driven take on the World Bank income categories.

### 5. Life Expectancy Gainers
Simple but striking: for each country, subtract 1952 life expectancy from 2007 life expectancy. Oman, Vietnam, and Indonesia top the list, each adding 35+ years — one of the great untold stories of the 20th century.

## Data Source

Built-in `plotly.express.data.gapminder()` — originally compiled by the [Gapminder Foundation](https://www.gapminder.org/). No download required; no API key needed.
